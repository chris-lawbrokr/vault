# Flow Templates — Bulk Page Editing

## Concept
Find all flows using a template and allow bulk-editing pages across funnels.

## Location
`https://app.lawbrokr.ca/libraries/pages` → `libraries/pages/` route

## Workflow
1. Find an existing page in the library
2. Give option to edit that page
3. Editing shows all funnels the page is linked to (pages are used in funnels)
4. Give option to apply edit to:
   - All funnels using this page
   - A specific subset of funnels
5. Creates a duplicate for any changed pages
